Berikut adalah Aplikasi Pemesanan Travel Berbasis Java Web (JSP)

Untuk mengaktifkan:
-copy file "vadimtravel.war" pada folder dist ke folder webapps server anda
-buat database "vadimtravel" pada database MySQL anda
-untuk menjalankan pada browser klik http://<nama-domain>/vadim-travel

Cara kerja:
-pemesan dapat langsung mengisi formulir pada halaman "Booking" atau melihat daftar tujuan pada halaman "Daftar Tujuan"
-Operator dapat memanipulasi pemesanan perjalanan dengan memasuki halaman http://<nama-domain/vadim-travel?controller=office , dengan tambahan Admin dapat memanipulasi para operator
